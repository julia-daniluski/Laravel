# Laravel

## O que é?

Um framework PHP, robusto que serve o padrão **MVC**:
**M** (como registra no banco) **V** (as páginas) **C** (requisições).

## Principais características

* Estrutura MVC;
* Roteamento simples;
* Conexão rápida com o BD (banco de dados);
* Sistema de autenticação/autorização.

## Estrutura de pastas

⇢ App

⇢ Route

⇢ Resources

⇢ Database

⇢ Public

⇢ Config
